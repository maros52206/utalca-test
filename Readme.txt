# utalca-test

La aplicación móvil desarrollada con el Framework Ionic se conecta a una API utilizando 
varios endpoints proporcionados. El objetivo principal de la aplicación es interactuar 
con los servicios ofrecidos por la API para obtener datos, enviar información y realizar 
otras operaciones según sea necesario.


Características Principales:
1-Conexión a la API: La aplicación se conecta a la API mediante los endpoints proporcionados para obtener y enviar datos
2-Funcionalidades de la Aplicación: La aplicación incluye diversas funcionalidades, como iniciar sesión, registrar usuarios, cambiar contraseña y visualizar listas de resultados de pruebas, todas las cuales interactúan con la API.
3-Interfaz de Usuario Intuitiva: La interfaz de usuario de la aplicación está diseñada para ser intuitiva y fácil de usar, lo que permite a los usuarios navegar fácilmente por las diferentes funcionalidades ofrecidas.
4-Seguridad: Se implementan medidas de seguridad para proteger la información del usuario, como la autenticación en el inicio de sesión y el registro de usuarios.
5-Compatibilidad Multiplataforma: La aplicación está desarrollada con Ionic, lo que garantiza su compatibilidad con múltiples plataformas, incluyendo iOS y Android.

Tecnologías Utilizadas:
Ionic Framework
Angular
HTML
CSS
TypeScript

## Instalación

1. Clona este repositorio en tu máquina local.
2. Instala las dependencias y/o paquetes del proyecto:
 npm install
3. Ejecute el comando
    ionic serve
